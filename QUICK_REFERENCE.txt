BUTTON NOT WORKING? DO THIS:
───────────────────────────────────────────────────────────────────────────────
STEP 1: Hard refresh
  - Windows: Ctrl + Shift + R
  - Mac: Cmd + Shift + R

STEP 2: Test isolation
  - Open test-reminder.html
  - Click "Show Modal"
  - Test buttons there
  - If buttons work = issue is Honey-specific
  - If buttons don't work = browser/system issue

STEP 3: Check browser console (F12)
  - Open Developer Tools
  - Go to Console tab
  - Look for RED error messages
  - Copy any errors

STEP 4: Manual test
  - Paste in console:
  - app.handleDismissReminder()
  - Should close modal immediately
